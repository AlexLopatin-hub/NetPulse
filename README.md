# NetPulse

[![Version](https://img.shields.io/badge/version-1.0.3-blue.svg)](https://github.com/alexlopatinhub/netpulse/releases)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Tauri](https://img.shields.io/badge/Tauri-2.0+-purple.svg)](https://tauri.app/)

Легковесный сетевой монитор для рабочего стола. Отображает текущий статус доступности нужных ресурсов в компактном окне, которое всегда остается поверх других приложений.
